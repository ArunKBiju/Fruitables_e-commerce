# 🛒 Fruitables – E-Commerce Frontend

A modern e-commerce interface designed with a focus on clean UI structure and precise layout implementation. This project demonstrates effective use of semantic HTML and professional styling frameworks.

### Live Preview
[https://arunkbiju.github.io/Fruitables_e-commerce](https://arunkbiju.github.io/Fruitables_e-commerce/)

### Project Overview
A fully styled frontend template for a fruit and vegetable e-commerce platform. The project focuses on creating a visually appealing user experience and structured content delivery.

* **Role:** Frontend Developer
* **Project Type:** Web Interface Development

### Tech Stack
* **HTML5:** Semantic structure and content organization.
* **CSS3:** Custom styling and layout design.
* **Bootstrap 5:** Component styling and UI framework integration.
